Project 1: Instructions

Due: 19 March

Description

An important part of programming well is knowing your audience, and programs are instructions that we ask computers to do for us. In this project you will write instructions for three different target audiences: a friend, a small child, and an alien from another planet.

Procedure

Start by choosing a multi-step task that you know very well such as tying your shoes or making a sandwich. Then, write the steps to complete that task as if you were going to tell one of your friends how to do it. Next, list the steps again with enough detail that a small child could understand it. Finally, write out the steps with an absurd amount of details, so that an alien from another planet could complete the task. For this final list you should imagine the alien you will be teaching, and write out the assumptions you have made about it (How well does it understand English? Does it have hands? How many?). You can choose to use an alien from pop culture such as Alf or the alien from Alien (though good luck teaching her how to tie shoes).

Grading

  • 0 - Nothing submitted.
  • 1 - You started a list.
  • 2 - You only submitted one list.
  • 3 - Your only submitted two lists.
  • 4 - You submitted three lists, but they lack some details.
  • 5 - You submitted three well-labeled and detailed lists, as well as a description of your alien.

Submitting

Write your three lists of increasingly detailed steps in a google doc and share it with me before class on Monday. Please clearly label the audience of each set of instructions, and include the assumptions you are making about the alien.

About

Class taught at Bentley School by Joey Knapp.

Course Description
This course introduces students to programming through the python programming language. Programming is a key element of working in computer science -- both studying CS in college, and in future career paths. Even students who do not intend to pursue a software engineering or technical profession will benefit from understanding how computers work and how to program. Students who pursue technical paths will be able to learn other programming languages much easier after learning python; programming principles are the same across the most commonly used languages. Students in this class work to recognize the sort of problems that programs and computers can solve. The major topics of this course are the computational thinking skills of decomposition, data representation, abstraction, automation, and algorithms. Understanding these topics makes programming easier and more efficient, and the programs that result from this understanding run better and are more robust. Skills in these areas will also be useful for many kinds of work as professionals of all sorts rely more and more on solving problems with technology to be productive in their fields.

Learning Objectives
  • Start, write, test, edit, and run python programs.
  • Use variables, conditionals, loops, functions, lists, and strings.
  • Decompose problems into smaller goals.
  • Abstract and generalize tasks to build robust programs.
  • Search deliberately for ways to automate aspects of their solution.
  • Design, implement, and analyze algorithms in their programs others.
  • Read and understand python code.
  • Troubleshoot syntactic and logical errors.
  • Use try/except blocks to avoid fatal errors.
  • Use python packages such as random and csv.
  • Document their code using comments.
  • Open a file and parse the contents.
  • Analyze a data set and draw conclusions.
  • Generate data visualizations using python.
  • Write a paper in the style of a scientific journal.
  • Practice metacognition as they write and talk about their problem solving process to make that process more efficient and repeatable.
